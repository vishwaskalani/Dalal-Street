# Dalal Street

Welcome to the *Dalal Street* project! This repository contains organized notes, daily logs, and scripts for financial market research. Built using MkDocs.

## Structure
- docs/: Markdown files for the site.
- scripts/: Python scripts to fetch stock/news data.
- data/: Data files.
- notebooks/: Analysis notebooks.

## Setup
- Install: `pip install mkdocs mkdocs-material requests`
- Serve site: `mkdocs serve`
